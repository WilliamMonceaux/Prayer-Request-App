# LASER-Kit
Linting, Alignment, and Standards Enforcement Repo

## Installation

### Option 1: Use GitHub UI Templates

When creating a new repo in the GitHub UI select the `IOD-LASER-Kit` as the template from which to create the new repo.

### Option 2: Use the CLI

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/IOD-SE/IOD-LASER-Kit/main/install.sh)"
```

### Option 3: Use the CLI and `gh`

```bash
gh repo create my-new-project --template IOD-SE/IOD-LASER-Kit --public --clone
```
